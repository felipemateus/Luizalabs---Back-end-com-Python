# Luizalabs---Back-end-com-Python
## Sobre o Bootcamp

Bootcamp intensivo de Back-end com Python oferecido pela Luizalabs e DIO, focado em desenvolvimento de aplicações robustas e escaláveis.

## Objetivo

Formar desenvolvedores Back-end capacitados para criar soluções completas utilizando Python e boas práticas de engenharia de software.

## Tecnologias

- Python 3.x
- Django / FastAPI
- PostgreSQL
- Docker
- Git & GitHub



## Requisitos

- Conhecimento básico em programação
- Familiaridade com linha de comando
- Vontade de aprender
